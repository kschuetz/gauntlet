## [Unreleased]

No changes yet

## [0.4.4] - 2021-01-21

### Added

- Add shrink strategies for strings and characters

### Changed

- Improve performance of `ShrinkCollection`

## [0.4.3] - 2020-10-02

### Added

- Add built-in arbitraries:
  - `alphaStrings`
  - `alphaUpperStrings`
  - `alphaLowerStrings`
  - `alphanumericStrings`
  - `identifiers`
  - `alphaCharacters`
    - `alphaUpperCharacters`
  - `alphaLowerCharacters`
  - `alphanumericCharacters`
  - `numericCharacters`
  - `punctuationCharacters`
  - `asciiPrintableCharacters`
  - `controlCharacters`

### Changed

- Upgrade Kraftwerk to 0.10.0
